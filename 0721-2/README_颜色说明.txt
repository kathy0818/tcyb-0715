二、橙底是什么 + 学生要做的事
------------------------------------------------------------------------

■ 橙底  (LaTeX 命令 \cck)
    = 【我先填好的参考数值】, 只是给学生一个起点, 不是最终测得值。

    橙底一共就两处(main 和 Rebuttal 各一份, 值要一致):

    (1) Fig.6 三个 per-image AIU: 现填参考值 Full=0.62 / High-Freq=0.45 / Low-Freq=0.41。
        参考值只保证 Full > High-Freq > Low-Freq 的相对关系, 具体数字以复算为准。

    (2) Fig.6 的 AIU 阈值集合: 现填参考值 T = {0.01, 0.02, ..., 0.99}。

    学生请用【原始 GT + 原始预测概率图 + 论文正式评测脚本】复算:
        与参考值一致 -> 把 \cck{...} 去掉, 定稿;
        与参考值不同 -> 用真实算出的值替换, 再去掉 \cck{...}。
    判断标准: 编译后的 PDF 里只要还有橙块, 就是数值还没最终确认。

    ⚠ 不能用 PDF/图片反算值。导师已明确否掉 0.79/0.61/0.57 和 0721-1 里的
      0.71/0.28/0.55, 所以橙色数值必须回到原始数据复核。
