一、三种底色分别代表什么(打开 PDF 就能看到)
------------------------------------------------------------------------

■ 绿底  (LaTeX 命令 \edt / edtblk)
    = 本轮【已按导师意见改好】的文字。DONE,无需再动。
      给你和导师核对"这一轮到底改了哪些字"。

■ 橙底  (LaTeX 命令 \cck / cckblk)
    = 本轮【先做的保守改法】,但改动依赖代码实现,
      需要学生【对照模型代码】确认后才能定稿。
      (已用最稳妥的写法规避方向性错误,确认后微调即可。)

■ 黄底  [[TO FILL ...]]  (LaTeX 命令 \FILLIN)
    = 【必须由学生用原始数据 + 正式评测脚本】计算或提供的空。
      ⚠ 不能用 PDF/图片反算值:导师已明确否掉 0.79/0.61/0.57
        和 0721-1 里的 0.71/0.28/0.55(High/Low 面板疑似错配,且不可复现)。
      判断标准:编译后的 PDF 里只要还有黄块,就是没填完。
