一、三种底色分别代表什么(打开 PDF 就能看到)
------------------------------------------------------------------------

■ 绿底  (LaTeX 命令 \edt / edtblk)
    = 本轮【已按导师意见改好、且已定稿】的文字。DONE, 无需再动。
      给你和导师核对"这一轮到底改了哪些字"。

■ 橙底  (LaTeX 命令 \cck)
    = 【我先填好的参考数值】(Fig.6 的三个 per-image AIU、AIU 阈值集合)。
      这些只是让学生有个起点, 不是最终测得值。
      学生请用【原始 GT + 原始预测概率图 + 论文正式评测脚本】复算一遍:
        与我填的值一致  -> 直接把 \cck{...} 去掉, 定稿;
        与我填的值不同  -> 用真实算出的值替换, 再去掉 \cck{...}。
      ⚠ 不能用 PDF/图片反算值。导师已明确否掉 0.79/0.61/0.57
        和 0721-1 里的 0.71/0.28/0.55, 所以橙色数值必须回到原始数据复核。

■ 黄底  [[TO FILL ...]]  (LaTeX 命令 \FILLIN)
    = 【只有学生知道、我无法代填】的样本标识: 数据集名 + sample ID。
      判断标准: 编译后的 PDF 里只要还有黄块, 就是没填完。
